message   ) 
 

Value:

        if (logger->isDebugEnabled()) {\
        ::log4cxx::StringBuffer oss; \
        oss << message; \
        logger->forcedLog(Level::DEBUG, oss.str(), __FILE__, __LINE__); }}
Logs a message to a specified logger with the DEBUG level.

Parameters:
logger the logger to be used.
